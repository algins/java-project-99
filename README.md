### Requirements:
* Git
* Make
* JDK 21
* Gradle 8.7

### Setup:
```sh
$ git clone https://github.com/algins/java-project-99.git && cd java-project-99
$ make setup
```

### Start development server:
```sh
$ make start-dev
```

### Run tests:
```sh
$ make test
```

### Run linter:
```sh
$ make lint
```
